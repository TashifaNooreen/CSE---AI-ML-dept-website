Value Added Courses

The Department of CSE AI & ML offers specialized value-added courses to enhance students' technical skills and prepare them for industry demands. These courses are designed to provide hands-on experience and in-depth knowledge. Upon completion, students receive a separate certificate from the college.

JavaScript

Learn the fundamentals of JavaScript, the backbone of modern web development. This course covers DOM manipulation, ES6 features, and building interactive web applications.

Ollama and LangChain

Dive into cutting-edge AI tools like Ollama and LangChain. This course focuses on building AI-driven applications and understanding natural language processing pipelines.

HTML & CSS

Master the basics of web design with HTML and CSS. Learn how to create responsive, visually appealing websites and understand modern layout techniques.

Full Stack Development

Become a full-stack developer by learning both front-end and back-end technologies. This course covers frameworks like React, Node.js, and database management.

Advanced Python Tools

Explore advanced Python libraries and tools such as Pandas, NumPy, and TensorFlow. This course is ideal for data science and machine learning enthusiasts.

Advanced Data Structures

Strengthen your problem-solving skills with advanced data structures like graphs, trees, and hash tables. This course is essential for competitive programming and software development.
